# example4

```text
두 개의 음이 아닌 정수 A와 B가 주어지면 숫자 A * B의 이진 표현에서 1로 설정된 비트 수를 반환한다. 

예를 들어, A = 3 및 B = 7일 때 함수는 3을 반환해야 합니다. 

왜냐하면 A * B = 3 * 7 = 21의 이진 표현은 10101이기 때문입니다. 
1로 설정된 3비트를 포함합니다. 
A와 B는 [0...100,000,000] 범위 내의 정수입니다. 
해결책에서는 정확성에 초점을 맞춥니다. 
솔루션의 성능은 평가의 초점이 되지 않습니다.
```

## 요약

> 단일파일로 러스트 실행 및 테스트

```sh
mkdir example4
cd example4
cargo init
cargo run
```
